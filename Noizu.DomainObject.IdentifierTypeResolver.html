    Noizu.DomainObject.IdentifierTypeResolver (Noizu.AdvancedScaffolding v1.2.23) View Source

    The Noizu.DomainObject.IdentifierTypeResolver module provides plumbing and default logic for built-in type handlers in the Noizu.DomainObject framework.

    This module acts as a resolver for identifier type handlers, allowing dynamic resolution of the appropriate handler for a given identifier type.
